# T3Test
Unreal engine learning project

How to play: 
weapons: 
1 - move   push/pull
2 - rotate clockwise/counterclockwise
3 - destroy block/figure

Destroying block charge weapons. The same color of block charge the same colored weapon.

Scores: 
1 layer - 1 score
2 layers - 3 scores
3 layers - 5 scores
4 layers - 7 scores
